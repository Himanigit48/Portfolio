My Projects

Wanderlust

Technology used: JS | NodeJS | ExpressJS | MongoDB | CSS

Developed a Full Stack Website called “Wanderlust”, a global accommodations booking website.

Integrated user authentication login/signup page, property listing, allowing users to easily upload and showcase images of their properties, browse edit, booking systems.

Implemented backend functionality using Node.js and Express.js for server-side development, along with MongoDB for database management

Integrated a review and rating system, enabling users to leave feedback and rate their stays, fostering a community-driven platform.

This project highlights my ability to combine multiple technologies to create a comprehensive and user-friendly platform.

Weather Website

Technology used: ReactJS | CSS

I developed a dynamic weather website using React that retrieves real-time weather data from an external API.

This website allows users to enter the name of any city and instantly receive up-to-date weather information.

By leveraging React's component-based architecture and state management, the website delivers a highly responsive and user-friendly experience.

This approach allows for efficient rendering of dynamic data, seamless user interactions, and a modular design that enhances maintainability and scalability.
